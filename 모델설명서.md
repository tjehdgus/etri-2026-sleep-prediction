# 모델 설명서 — ETRI 2026 휴먼이해 AI 수면 7지표 예측

## 1. 과제 및 데이터
스마트폰·웨어러블 센서 라이프로그(전날 하루)로 그날 밤 수면의 **7개 이진 지표**를 예측한다.
- **Q1~Q3** (설문): 수면질/피로/스트레스 — 개인 전체기간 평균 대비 이진화
- **S1~S4** (Withings 수면센서): TST/SE/SOL/WASO의 NSF 가이드라인 준수 여부 (절대기준)
- 평가지표: **Average (Macro) Log Loss** → 보정된 확률 출력이 핵심
- train 450 / test 250, 피험자 10명(id01~id10), 시간적 분포 이동(test가 train 후반·이후 날짜)

## 2. 전체 구조 (단일 노트북 `solution.ipynb`)
원본 데이터(`./data/`)만으로 전체를 학습/추론. 외부 산출물 불필요. 실행시간 약 12분.

### 2.1 센서 피처
12개 parquet을 (피험자, lifelog_date) 단위로 집계:
- 시간대별(all/day/eve/prebed/night/morning/deep) 통계(mean/std/min/max/median/sum)
- 피험자내 시계열 파생(diff/rolling/EMA/expanding) — 분포이동 대응
- 야간 윈도우는 lifelog_date 기준 정렬

### 2.2 베이스 소스 (9종)
| 소스 | 설명 |
|---|---|
| tree | LGBM+CatBoost+ExtraTrees 앙상블 + pseudo-labeling, ExtraTrees 중요도 피처선택 |
| prior | 피험자별 라벨 사전확률(개인 기저율) |
| GRU/BiLSTM/TCN/Attn | 1D-CNN + 시퀀스 인코더 + 피험자 임베딩, 멀티태스크 7헤드. **다중 seed 평균(SWA식 분산감소=DLens)** |
| SigLIP-Large | 센서를 24×채널 이미지화 → `google/siglip-large-patch16-256` frozen 임베딩 → LGBM |
| SleepTree2 | 야간 움직임/심박/충전으로 주수면 구간 재구성(TST/SE/SOL/WASO 추정) → tree |
| Neighbor | 같은 피험자 날짜인접 라벨의 지수가중 평균 |

### 2.3 블렌딩 + 잔차 보정
1. **per-target greedy coordinate-descent** 블렌딩 (prior 앵커)
2. **DSQ12**: daystate(30분 bin 비지도 NMF 일상상태) 기반 Q1/Q2 라우팅 (residual-alpha)
3. **pairwise ranking**: 같은 피험자 (양성일,음성일) 쌍의 피처차로 순위학습(ridge logistic) → Q1/Q2 잔차. "개인 평균 대비" 라벨 정의에 구조적으로 부합
4. **forward-CV**: test의 시간이동을 반영해 forward-chaining CV로 S1/S4 블렌드 가중치 재선택(50% 강도)

## 3. 핵심 아이디어
- **개인 상대성**: 모든 피처에 피험자내 정규화. Q는 pairwise 순위학습으로 "개인평균 대비" 정의에 직접 대응
- **분산 감소**: 소규모(N=450)에서 DL 다중 seed 평균(DLens)이 안정적 일반화
- **시간이동 인지**: random-fold CV 대신 forward-chaining CV로 S1/S4 가중치 선택(분포이동 대응)
- **다중 모달 다양성**: 트리/시퀀스/비전(SigLIP)/수면재구성/이웃 등 서로 다른 관점 결합

## 4. 실행 환경
- OS: Windows-10 / Python 3.11.15
- numpy 2.4.6, pandas 2.3.3, scikit-learn 1.7.2, lightgbm 4.6.0, catboost 1.2.10, torch 2.11.0+cu128, transformers 4.57.6, timm 1.0.27

## 5. 사용한 공식 공개 사전학습 모델
- **SigLIP-Large** (`google/siglip-large-patch16-256`)
  - 출처: https://huggingface.co/google/siglip-large-patch16-256
  - 용도: frozen 비전 임베딩 추출(학습 안 함). transformers 라이브러리로 자동 다운로드.
